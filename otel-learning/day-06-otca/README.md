# Day 6 - OTCA Concepts

## Overview

Day 6 connects the hands-on work in this repository to OTCA-style learning goals. The focus is on practical understanding instead of memorization alone.

## Learning Outcomes

- Explain the core OpenTelemetry data types
- Describe the role of the Collector in a telemetry pipeline
- Understand resource attributes, context propagation, exporters, processors, and sampling

## Study Map

- Traces and spans: Day 1 and Day 2
- Context propagation: Day 2 API instrumentation
- Metrics and log signals: Day 2 and Day 4
- Collector architecture: Day 3
- Exporters and pipelines: Day 4 and Day 5

## Troubleshooting

- If a concept feels abstract, go back to the matching day and run the example again.
