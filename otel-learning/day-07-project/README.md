# Day 7 - Real-World Mini Project

## Overview

Day 7 brings the pieces together with small project-style examples that feel closer to real application flows.

## Learning Outcomes

- Run an OTLP trace example against the Collector
- Expose service metrics for scraping
- Practice thinking in terms of user journeys and service behavior
