Quick Start

Get up and running in under 5 minutes. Clone the repo, install Python dependencies, and run your first tracing example.

What You Need

🐍
Python 3.8+
The examples use Python with the OpenTelemetry SDK.
🐳
Docker Desktop
Required for Day 3+ when running the OTel Collector.
📝
Code Editor
VS Code recommended, but any editor works.

Step-by-Step Setup

1

Clone the repository

shell
git clone https://github.com/ThejaReddy1/learn-otel.git
cd OTelInstrumentation
2

Install Python dependencies

powershell
py -m pip install -r .\shared-resources\python\requirements.txt
3

(Optional) Install auto-instrumentation packages

powershell
py -m pip install -r .\shared-resources\python\requirements-auto.txt
4

(Optional) Start the Collector

Needed for Day 3+ examples that export via OTLP.

powershell
docker compose -f .\shared-resources\collector\docker-compose.yaml up -d
5

Run your first example!

powershell
py .\otel-learning\day-01-basics\manual-instrumentation\payment.py

You should see span data printed to the console. 🎉

Common Issues

Try python or python3 instead. On some systems, the Python launcher isn't installed.

Make sure the Collector is running on localhost:4317 (gRPC) and localhost:4318 (HTTP). Check with docker ps.

Keep the app running and check http://localhost:8000/metrics. The metrics endpoint only exists while the process is alive.
