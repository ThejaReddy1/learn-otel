Learning Path

Your 8-Day Journey

Each day is self-contained. Start from Day 0 and build up, or jump to what you need.

D0
Foundations
Build a mental model for observability before touching any code. Understand logs, metrics, traces, and where OpenTelemetry fits.
Observability Three Signals Mental Model
D1
OpenTelemetry Basics
Create your first tracer, open a span, and see code execution become a timeline. Learn resource attributes and metadata.
Manual Instrumentation Resource Attributes
D2
Instrumentation
Trace real HTTP calls, log meaningful events, and count what your application does. Manual, library, and auto approaches.
API Library Auto Metrics Logging
D3
Collector Deep Dive
The Collector is the control point between your code and backends. Learn receivers, processors, exporters, and pipelines.
Docker Receivers Pipelines
D4
Exporters & Backends
Connect your telemetry to the world. OTLP for traces and logs, Prometheus for metrics, and trace-log correlation.
OTLP Prometheus Correlation
D5
Pipelines & Processors
Batching improves efficiency. Sampling reduces cost. Span processors decide how telemetry flows from creation to export.
Span Processors Sampling
D6
OTCA Concepts
Map certification concepts back to hands-on examples. Resource attributes, exporters, processors, context propagation — all connected.
Study Map Cert Prep
D7
Mini Project
Bring it all together. Trace a shopping cart flow, expose fitness metrics, and practice thinking in user journeys.
Shopping Cart Fitness Service
